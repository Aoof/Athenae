# Athenae Initiative
The official repository of Athenae's website. Built with care using Svelte 5, TypeScript, and TailwindCSS. Every line of code is human-made, and AI use is purely consultational. Therefore, all assets are also human-made.

## Who are we?
A Non-profit initiative that focuses on battling the rise of anti-intellectualism. Providing free and fun ways to encourage practicing humanities.

## Features
- Responsive - The design is optimized for all devices
- Dark/Light Mode - Different themes with preference detection
- Bilingual - The website is fully translated and toggleable between french and english
- MDsveX - Write content with markdown files 

## Stack
- Framework: SvelteKit and Svelte 5
- Styling: TailwindCSS
- Language: TypeScript
- Content: MDsveX
- Icons: Lucide Icons

## Content
Ideally this website will mutate into a CMS, but for now all content will be managed through the [content directory](src/lib/content/)

## 3rd party apps
- Donations: CanadaHelps
- Email: SendGrid
  - Volunteer Form (name, email, interests, message, resume)
  - Content Submission Form (type, title, author, content, attachments)
  - Subscriber Form (Email, language)
